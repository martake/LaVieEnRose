# v0.2修正点

- 「探索ボーナス」の概念を削除してください。どれだけ未知の状態を探索したかではなく、純粋に「得られた報酬」のみを評価します
- 「予測する報酬の差」に対して「得られた報酬」が小さい状態が続いている場合、また、「得られる報酬」が小さい状態が続いている場合は、エージェントは脅威の状態にあります。
  脅威の状態にあることは、「世界」への予測が誤っていることを示します。その脅威の大きさに応じ、世界のパラメータを大きく更新してください。
  これにより「世界の再探索」が促進されます。
- エージェントが画面外に飛び出す、または画面の端ギリギリに貼り付く場合があります。エージェントが移動可能な範囲は、画面内の中央付近のみに制限してください。
